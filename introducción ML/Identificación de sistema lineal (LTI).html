Ejemplo de Identificación de Sistemas¶

Ignacio D√≠az Blanco, 2019. Universidad de Oviedo.

En este ejemplo se muestra el proceso de identificaci√≥n de un sistema sencillo mediante minimizaci√≥n cuadr√°tica utilizando funciones de la librer√≠a sklearn. Algunas variantes que pueden experimentarse para observar los resultados son las siguientes:

  • modificar el sistema de test,
  • variar el ruido a√±adido
  • o modificar el par√°metro $\alpha$ de regularizaci√≥n.

Generación de datos de entrenamiento¶

En este bloque excitamos el siguiente sistema din√°mico:

$$ G(z) = \frac{1+2z^{-1}}{1-z^{-1}+0.9z^{-2}} \qquad \longleftrightarrow \qquad y_k = u_k + 2 u_{k-1} + y_{k-1} - 0.9 y_{k-2} $$

con una se√±al de excitaci√≥n aleatoria (ruido blanco gaussiano). La excitaci√≥n mediante ruido blanco es muy rica en informaci√≥n; excita todos los modos del sistema, lo que hace que el problema minimo-cuadr√°tico est√© bien condicionado y no de lugar a soluciones degeneradas. A la capacidad de una se√±al para excitar un sistema y producir soluciones bien condicionadas se la denomina persistencia en la excitaci√≥n.

Planteamos el problema en términos matriciales¶

Planteamos el problema de forma equivalente, pero en t√©rminos matriciales, para adecuarlo a su resoluci√≥n mediante los m√©todos m√≠nimo-cuadr√°ticos. La matriz $X$ de regresores, contendr√° valores de la entrada y de la salida con distintos retardos, de forma que el producto de $X$ por una matriz $T$ de coeficientes del modelo den lugar a la salida en el instante $k$, resultando en un problema de m√≠nimos cuadrados.

$$ y_k = T X $$
In [ ]:
def tddata(uk,yk,AR,MA):
    Q = len(yk)
    X = []
    y = []
    
    idxu = arange(0,MA+1)
    idxy = arange(1,AR+1)
    # print idxu, idxy
    for k in range(AR+1,Q):
        X.append(hstack((
                    uk[k-idxu],
                    yk[k-idxy]
                )).tolist()  )
        y.append(yk[k].tolist() )
    return X,y

X,y = tddata(uk,yk,2,1)

Xtst,ytst = tddata(utst,ytst,2,1)

X = array(X)
y = array(y)
Xtst = array(Xtst)
ytst = array(ytst)

Solución empleando la librería sklearn¶

# NORMALIZACI√ìN DE LAS VARIABLES
from sklearn.preprocessing import MinMaxScaler

# Creamos una clase de normalizaci√≥n
minmaxx = MinMaxScaler(feature_range=(-1,1))

# Aplicamos m√©todo fit para calcular la normalizaci√≥n (hallar m√°ximos y m√≠nimos)
minmaxx.fit(X)

# Aplicamos la normalizaci√≥n a los datos de entrenamiento y a los de test
X_norm    = minmaxx.transform(X)
Xtst_norm = minmaxx.transform(Xtst)

# Creamos una clase de normalizaci√≥n para las salidas
minmaxy = MinMaxScaler(feature_range=(-1,1))
