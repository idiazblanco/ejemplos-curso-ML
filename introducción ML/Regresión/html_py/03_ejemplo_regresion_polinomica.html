Descripcin - Ejemplo de regresin polinmica. Se utiliza la regresin multivariable empleando un vector de regresores compuesto por los monomios del polinomio a aproximar $\varphi = [1, x, x^2, x^3, \dots, x^n]^T$ y los coeficientes de dichos monomios son los parmetros a estimar $\theta = [a_0, a_1, a_2, \dots, a_n]^T$

Este ejemplo puede adaptarse fcilmente al caso ms general de aproximacin modelos no lineales como combinacin lineal de trminos no lineales $$ y = f(x,\theta) \approx a_1 f_1(x) + a_2 f_2(x) + \dots a_n f_n(x) $$ donde $f$ es una funcin no lineal desconocida y $f_i$ son trminos no lineales que se pueden elegir convenientemente en cada problema, aprovechando el conocimiento disponible. Por ejemplo se pueden elegir funciones trigonomtricas, polinmicas, o de otro tipo, si se sabe que concuerdan con la fsica del proceso. En este caso, para aproximar un polinomio $y = a_0 + a_1 x + a_2 x^2 + a_3x^3$ se ha elegido $$ f_1 = x, \quad f_2 = x^2, \quad f_3 = x^3 $$ El trmino independiente $a_0$ lo estima el propio modelo en su parmetro intercept_.

In[1]:
import numpy as np
import matplotlib.pyplot as plt


# Nmero de datos del ejemplo
N = 3000;

# Generamos datos acordes con el modelo polinmico
a3, a2, a1, a0 = 1, -5, 1, 2
epsilon = 2.5*np.random.randn(N);

x = np.random.randn(N);
y =  a0 + a1*x + a2*x**2 + a3*x**3 + epsilon; 

# creamos una matriz con los trminos no lineales del polinomio
X = np.column_stack((x, x**2, x**3))
In[2]:
from sklearn.linear_model import Ridge
modelo = Ridge(alpha=0.0001)

modelo.fit(X,y)

print(f'coeficientes          = {modelo.coef_}')
print(f'trmino independiente = {modelo.intercept_}')

coeficientes          = [ 1.19645724 -5.05424567  0.95337915]
trmino independiente = 2.1066022950773715

# generamos datos de test nuevos
xtest = np.linspace(-3,3,1000)
ytest = a3*xtest**3 + a2*xtest**2 + a1*xtest + a0

# construimos la matriz X con los mismos trminos no lineales que para el entrenamiento
Xtest = np.column_stack((xtest,xtest**2,xtest**3))
ypred = modelo.predict(Xtest)

plt.figure(2,figsize=(7,7));
plt.plot(x,y,'.',alpha=0.3);			# datos de entrenamiento
plt.plot(xtest,ypred,'r',linewidth=2);  # prediccin 
plt.plot(xtest,ytest,'g',linewidth=2);  # modelo ideal
plt.xlabel('$x, x_{test}$')
plt.ylabel('$y, y_{test}$')
plt.legend(('entrenamiento','prediccin','real'))
plt.grid(True)
plt.axis(xmin=-3,xmax=3,ymin=-50,ymax=50)
plt.show()
