Reduccin de la dimensionalidad (datos de motor de laminacin)

Ignacio Daz Blanco, 2019. Universidad de Oviedo

En este ejemplo se aplican dos tcnicas de reduccin de la dimensionalidad

  • Anlisis de componentes principales (PCA), tcnica lineal que implica la proyeccin de los datos sobre un hiperplano (en este caso 2D) definido por los vectores principales (principales direcciones de variabilidad de los datos).
  • t-stochastic neighbor embedding (t-SNE), es una tcnica no lineal, que permite obtener un conjunto de puntos en un espacio latente de baja dimensin (en este caso 2D) de forma que, probabilsticamente, preserven las relaciones de vecindad entre ellos

Las tcnicas de reduccin de la dimensionalidad detectan la estructura latente de baja dimensin en el espacio de los datos lo que les permite obtener una representacin de los datos multidimensionales empleando un nmero reducido de variables latentes sin prdida significativa de informacin.

Si se elige un nmero convenientemente reducido de variables latentes (2 o 3) es posible representar visualmente el proceso mediante dichas variables, creando de esta manera mapas del proceso que permitern interpretarlo.

En este ejemplo, se incluye una extraccin de 4 caractersticas a partir de las corrientes de inducido de un motor de laminacin que son proyectadas mediante PCA y tSNE en espacios latentes de 2D y posteriormente visualizadas, mostrando los distintos estados del proceso (regeneracin, laminacin, etc.)

In[1]:
import numpy as np
import matplotlib.pyplot as plt
from scipy.io.matlab import loadmat
In[2]:
datos = loadmat('../../datos/datosmotorlaminacion.mat')

fm = 5000       # Frecuencia de muestreo
tm = 1/fm       # Periodo de muestreo


varNames = [i[0] for i in datos['nombres'][0]]


# OBTENEMOS INFORMACIN DE LOS DATOS IMPORTADOS
p = datos['p']       # matriz de datos (4 filas, muchas columnas)
Q = p.shape[1]       # tamao de la muestra
N = 1000             # tamao de las ventanas en la extraccin de caractersticas
L = 300              # solapamiento entre ventanas en la extraccin de caractersticas
t = np.arange(Q)*tm     # vector de tiempos 

# TRAZAMOS LAS VARIABLES
plt.figure(1,figsize=(15,10))
for k in range(4):
    plt.subplot(4,1,k+1)
    plt.plot(t,p[k,:])
    plt.ylabel("$"+varNames[k]+"$")
    plt.xlabel('t (seg)')
plt.tight_layout()

Extraccin de caractersticas

  • Realizamos un enventanado, con ventanas de tamao N solapadas, separadas L muestras entre ellas
  • para cada ventana, obtenemos 4 caractersticas: la media, la desviacin tpica, la fraccin de muestras para las que la corriente de inducido $I_a$ supera 0.5 p.u y el valor RMS de la banda $300 \pm 20$ Hz
In[3]:
# extraeremos caractersticas de la corriente de inducido
y = p[0,:]


tF= []    # lista de tiempos para las caractersticas
F = []    # lista con las caractersticas por ventanas

i = 0
for k in range(N,Q,L):       # desde N hasta Q, a saltos de L en L...
    yv = y[k-N:k]/9000        
    Y  = np.abs(np.fft.fft(yv))**2/N
    f = np.arange(0,fm,fm/N)
    idx = np.where((f>280)&(f<320))
    F.append([
        np.mean(yv),
        np.std(yv),
        np.sum(yv>0.5)/float(N),
        np.sqrt(sum(2*Y[idx])/float(N))])
    tF.append(k*tm)
    i = i + 1
X = np.array(F)
tX= np.array(tF)

# Etiquetas de las caractersticas recin obtenidas
nombresCaracteristicas = [
    r'media $I_a$', 
    r'desv. tpica $I_a$',
    r'fraccin > 0.5 p.u.', 
    r'RMS  en banda $300 \pm 20$ Hz']

Normalizacin de variables

Normalizamos las variables a $\pm 1$

In[4]:
# NORMALIZACIN DE LAS VARIABLES
from sklearn.preprocessing import MinMaxScaler

# Creamos una clase de normalizacin
minmaxx = MinMaxScaler(feature_range=(-1,1))

# Aplicamos mtodo fit para calcular la normalizacin (hallar mximos y mnimos)
minmaxx.fit(X)

Xnorm = minmaxx.transform(X)

Proyeccin PCA

Proyectamos los datos en el plano definido por los vectores principales y que atraviesa el centro de masas (vector de medias $\mu$) de los puntos

In[5]:
from sklearn.decomposition import PCA

pca = PCA(n_components=2)
pr = pca.fit(Xnorm).transform(Xnorm)

plt.figure(1,figsize=(15,3))

for k in range(4):
    plt.subplot(1,4,k+1)
    plt.scatter(pr[:,0],pr[:,1],c=X[:,k],linewidths=0)
    # plt.title(u'caracterstica %d'%k)
    plt.title(nombresCaracteristicas[k])

Proyeccin tSNE

Proyectamos los datos utilizando t-stochastic Neighbor Embedding (tSNE), una tcnica de manifold learning

In[6]:
from sklearn.manifold import TSNE


tsne = TSNE(n_components=2,perplexity=70, method='exact')

pr = tsne.fit_transform(Xnorm)

plt.figure(1,figsize=(15,3))

for k in range(4):
    plt.subplot(1,4,k+1)
    plt.scatter(pr[:,0],pr[:,1],c=Xnorm[:,k],linewidths=0)
    plt.colorbar()
    # plt.title(u'caracterstica %d'%k)
    plt.title(nombresCaracteristicas[k])
